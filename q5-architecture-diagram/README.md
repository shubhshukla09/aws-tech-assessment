# Task 5: Architecture Diagram

This directory contains the architecture diagram for the AWS infrastructure created in this assessment.

## Diagram Overview

The architecture represents a highly available web application with the following components:

- **Frontend**:
  - CloudFront CDN
  - S3 Bucket for static assets
  - Web Application Firewall (WAF)
  - Route 53 for DNS

- **Application Layer**:
  - Application Load Balancer (ALB)
  - Auto Scaling Group (ASG) with EC2 instances
  - Security Groups for network security

- **Data Layer**:
  - Amazon RDS (Multi-AZ)
  - ElastiCache (Redis)
  - S3 for storage

- **Monitoring & Management**:
  - CloudWatch Alarms & Logs
  - AWS Config
  - CloudTrail

## How to View/Edit the Diagram

1. **Using draw.io (Recommended)**:
   - Go to [draw.io](https://app.diagrams.net/)
   - Click on "File" > "Open From" > "Device..."
   - Select `architecture-diagram.drawio` from this directory

2. **Exporting the Diagram**:
   - Open the diagram in draw.io
   - Click on "File" > "Export as" > "PNG" (or your preferred format)
   - Save the file as `Shubh_Shukla_Architecture_Diagram.png`

## Diagram Files

- `architecture-diagram.drawio` - Editable diagram file
- `Shubh_Shukla_Architecture_Diagram.png` - Exported PNG (generated by user)
